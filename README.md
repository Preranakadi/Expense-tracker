# Expense Tracker Dashboard

A modern web-based finance dashboard that allows users to record income and expenses, analyze spending patterns, and manage financial data efficiently

---

## Features

- **Add Transactions** – Enter an amount, select a category, choose Income or Expense, and add optional notes.
- **Live Summary Cards** – Balance, Total Income, and Total Expenses update instantly after every submission.
- **Transaction History Table** – Displays every transaction (newest first) with date, description, category, and amount.
- **Colour-coded Amounts** – Income shown in green, expenses in red.
- **Persistent Storage** – Transactions are saved to `localStorage` so data survives page refreshes.
- **Empty State** – Friendly "No transactions yet" message when the list is empty.
- **Responsive Layout** – Works on desktop, tablet, and mobile screens.

## How to Run Locally

1. Clone or download this repository.
2. Open `indexET.html` in any modern web browser.  
   No server, build step, or dependencies required.

---

## Deployment (GitHub Pages)

1. Go to the repository on GitHub.
2. Click **Settings** → **Pages**.
3. Under **Source**, select **Branch: main** and folder **/ (root)**.
4. Click **Save**.
5. After a few moments your site will be live at:  
https://your-username.github.io/Expense_tracker2/
